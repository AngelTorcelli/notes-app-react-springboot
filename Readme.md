# NoteCraft
Desarrollado por Angel Torcelli. <hr>

## Descripción
Esta es una aplicación de notas que permite a los usuarios crear, editar y eliminar notas. La aplicación está desarrollada con React y Spring Boot.

## Tecnologías
- Frontend
    - React (VITE)
    - HTML
    - CSS

- Backend
    - Spring Boot
    - MySQL
